Internet Explorer

  1. Click the "Accept" button.

  2. You will be prompted to either run or save the file. Choose "Run".

  3. Another window will open asking for confirmation to run the file. Choose "Run".

  4. A window will appear on the screen showing that the installation in proceeding. When installation is complete, the window will automatically close.

Firefox

  1. Click the "Accept" button.

  2. When prompted, save the file.

  3. Go to default save location and locate the file. Double-click the file.

  4. Another window will open asking for confirmation to run the file. Choose "Run

  5. A window will appear on the screen showing that the installation in proceeding. When installation is complete, the window will automatically close.
